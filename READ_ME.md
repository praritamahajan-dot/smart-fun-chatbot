Smart & Fun Chatbot



Smart \& Fun Chatbot is a full-stack web application that combines study assistance and entertainment in a single platform.

The project provides AI-powered study tools, an interactive chatbot, games, and user authentication, all running on a local server environment.


This project is designed to run locally using XAMPP and demonstrates practical use of frontend and backend technologies.



---



Features

* User authentication system (Sign Up \& Login)

* Secure storage of user data using MySQL

* AI-powered chatbot for interactive conversations

* Text summarization feature for study notes

* Multiple interactive games:
1. Quiz Game
2. Guess the Number
3. Rock Paper Scissors

* Clean, responsive, and user-friendly interface



---



Technologies Used

* HTML

* CSS

* JavaScript

* PHP

* MySQL

* XAMPP (Apache \& MySQL Server)



---



How to Run the Project Locally



1. Install XAMPP on your system.
   
2. Copy the project folder into the htdocs directory.
   
3. Start Apache and MySQL from the XAMPP Control Panel.
   
4. Create and import the required MySQL database (for user authentication).
   
5. Open your browser and visit:

http://localhost/smart-fun-chatbot



---



Important Note



* This project runs on a local server environment using XAMPP.



* It cannot be deployed directly on GitHub Pages because it uses PHP and MySQL.



* API keys have been removed from this repository for security reasons.
